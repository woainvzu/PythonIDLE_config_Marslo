PythonIDLE_config_Marslo
========================

The configuration of Python IDLE
## Usage:
# Linux:
Copy the .idlerc folder to $HOME, and re-start IDLE
# Windows:
Copy the .idlerc folder to %HOMEPATH%, and re-start IDLE

## Specific Shortcuts:
- Ctrl+p: Previous history
- Ctrl+n: Next history
- Alt+p:  Print page
- Alt+n:  New page
